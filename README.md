# 🎲 Dice Roller

A simple and responsive Dice Roller web application built using **HTML**, **CSS**, and **JavaScript**. Click the **Roll** button to simulate rolling a six-sided die and generate a random number between **1** and **6**.

## ✨ Features

- 🎲 Simulates rolling a six-sided die
- 🎯 Generates a random number from **1** to **6**
- 💬 Displays the result instantly
- 🎨 Clean and modern user interface
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/dice-roller.git
```

2. Navigate to the project folder.

3. Open `index.html` in your preferred web browser.

## 🎮 How to Use

1. Open the application in your web browser.
2. Click the **Roll** button.
3. A random number between **1** and **6** will be generated.
4. The result will be displayed below the button.
